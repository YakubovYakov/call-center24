Плановое лечение без направления - иногородним
Пациентам из тех регионов, с которыми подписаны соглашения - не требуется направление по форме 057/у-04. Направление выписывается непосредственно в КДО МО ДЗМ.
Для всех остальных регионов - направление является обязательным условием для плановой госпитализации.
