телефон Консультативно-Диагностического Отделения перинатального центра

8 495 612 39 56

Консультативно-диагностическое отделение перинатального центра находится по адресу 4-й вятский переулок, дом 39, на 2-ом этаже, регистратура - кабинет №8

График работы: понедельник-пятница с 9.00до 17.00.

Приём ведут врачи специалисты по предварительной записи:

Акушер-гинеколог ежедневно с понедельника по пятницу с 9.00 до 17.00.

Эндокринолог - по вторникам, с 9.00 до14.00

Офтальмолог – по понедельникам и средам с 15.30 до18.30

Также, по предварительной записи, осуществляемой женской консультацией, вы можете пройти пероральный глюкозо-толерантный тест (ПГТТ)

Для приёма по ОМС необходимы следующие документы:

1.Направление из женской консультации;

2. Паспорт;

3. Полис ОМС;

4. Снилс;

5. Обменная карта

При отсутствии направления и полиса ОМС вы можете обратиться в отделение платных Медицинских услуг по телефону 8 495 613 24 45

 

Консультативно-Диагностическое отделение (КДО)
